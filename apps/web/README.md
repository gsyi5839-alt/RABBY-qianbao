# Rabby Web 前端

基于 Rabby 扩展能力构建的 Web 端钱包应用。

## 技术栈（规划）

- React 18+ / Next.js 或 Vite
- TypeScript
- 状态管理：Redux / Zustand
- UI：Ant Design / Tailwind
- 与 packages/shared 共享核心逻辑

## 功能范围

- 账户创建/导入
- 多链资产管理
- Swap / Bridge / Gas 充值
- Dapp 连接（WalletConnect）
- 交易签名与审批

## 开发

```bash
yarn install
yarn dev
```
