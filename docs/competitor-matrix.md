# Wallet Competitor Matrix (Web)

Date: 2026-02-12
Scope: 10 wallet official sites + official help content

## Matrix (10 wallets)
| Wallet | Positioning | Officially stated capabilities |
| --- | --- | --- |
| MetaMask | Swap/Bridge/Portfolio as primary entry points | Swaps + Bridge + Portfolio dashboard |
| Coinbase Wallet | Self-custody and security messaging | Self-custody, permission/security guidance, DApp connection via QR/WalletLink |
| Trust Wallet | Non-custodial multi-chain wallet | DApp browser, swap, NFT, staking |
| Phantom | Multi-chain wallet | Multi-chain (Solana/Ethereum/Polygon), swaps, scam/spam token warnings |
| Rainbow | Mobile-first EVM wallet | Swap/bridge via RainbowRouter |
| OKX Wallet | Multi-chain Web3 portal | Non-custodial, DEX aggregation, DApp discovery |
| imToken | Multi-chain wallet | Multi-chain, desktop DApp browser, WalletConnect 2.0, security messaging |
| Exodus | Consumer-friendly wallet | Swap/stake, DApp access |
| SafePal | Cross-platform wallet | Swap + DApp connection |
| Bitget Wallet | Web3 discovery focus | DApp browser, staking/earn messaging |

## Cross-competitor patterns
- DApp discovery + security signals are consistently highlighted.
- Swap + Bridge are converging into a unified entry.
- Multi-chain support is now a default expectation.
- Security messaging emphasizes self-custody and permission control.

## Deep-dive notes (5 wallets)
- MetaMask: Swap/Bridge are primary conversion points; Portfolio dashboard is a centralized asset overview entry.
- Coinbase Wallet: Security narrative stresses self-custody and permission/risk guidance; DApp connection is emphasized.
- Trust Wallet: DApp browser and security scan messaging are key differentiators.
- OKX Wallet: Multi-chain + DEX aggregation + DApp discovery are core pillars.
- imToken: Desktop DApp browser + WalletConnect 2.0 + security messaging.

## Recommendations for Rabby
P0
- DApp browser with security risk labels and scam warnings.
- WalletConnect/QR flow standardized with permission prompts.

P1
- Unified Swap + Bridge entry.
- Multi-chain asset overview and quick network switching.

P2
- Staking/Earn hub with educational content and risk guidance.
